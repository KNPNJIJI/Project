Требования к проекту

Стек технологий, обязательный для применения слушателем, претендующим на сертификат:
• Java 11Spring
• Hibernate
• Система сборки
• Git
• Реляционная база данных
Требования, обязательные для применения слушателем, претендующим на сертификат:
• Мультимодульный проект
• Веб страницы c пагинацией через БД
• REST Controller - построение бэкэнда с применением различных HTTP методов, передачей
json объектов в теле запроса, правильное построение url
• Авторизация и аутентификация (пользователь и администратор)

Опционально (для слушателей, которые хотят больше):
• Spring Boot
• Использование статического анализатора кода
• Миграция базы данных с помощью liquibase/flyway/etc
• Контейнеризация
• Автоматическая сборка проекта (GitHub Actions
• Автотесты
• Автоматизация развёртывание в облаке
Оценки от 7 до 10 выставляются в зависимости от количества ошибок после финального кодревью преподавателем.

Реализованно:
Это приложение для сервиса аренды автомобилей.

URL приложения http://localhost:8080/carrental/
URL для администрирования приложения http://localhost:8080/carrental/admin-console/

REST endpoints
http://localhost:8080/carrental/api/cars/brand=all/ - отобрасить все авто
http://localhost:8080/carrental/api/cars/brand={brand}/ - за определённым брэндом
http://localhost:8080/carrental/api/cars/brand={brand}/model={model}/ за определённым брэндом + модель
Так же с датами
Например 
http://localhost:8080/carrental/api/cars/startRent=2023-01-23/endRent=2023-01-23/brand=all/

Так же в проекте на frontend-е был использован плагин DataTables (https://datatables.net/)
Миграция базы данных с помощью liquibase
